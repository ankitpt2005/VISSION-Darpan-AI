### 📝 Project Overview
**DARPAN AI** is a next-generation civic engagement platform designed to bridge the gap between citizens and local governance. By leveraging AI-driven analytics, we transform scattered public complaints into actionable insights, ensuring that every reported issue follows a strict path toward resolution.

---

### ✨ Key Features (Functional Prototype)
* **🌐 Branded Civic Feed:** A high-performance social interface for citizens to report, upvote, and track local issues in real-time.
* **📍 Live AI Heatmap:** An interactive geospatial dashboard utilizing Tailwind animations to visualize "Problem Hotspots" based on complaint density.
* **🏆 Gamified Leaderboard:** A "Civic Heroes" system that incentivizes public participation through points and rankings, fostering community spirit.
* **📱 Responsive Architecture:** A sleek, glassmorphic UI optimized for both desktop governance dashboards and mobile citizen reporting.

---

### 🧠 The AI Engine (Core Logic)
* **🤖 Intelligent Clustering:** Uses **K-Means Clustering** to group duplicate complaints into singular "Mega Hotspots," optimizing administrative resources.
* **📄 NLP Accountability:** Matches reports with political manifestos using **Natural Language Processing** to track electoral promises.
* **⏳ 7x7 Escalation Matrix:** An automated logic that escalates unresolved issues through 7 levels of hierarchy every 7 days.
